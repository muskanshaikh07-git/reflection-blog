In this unit we are learning about typebot which is an outstanding topic for me because here we are doing the hand-on presentation. Unlike any theoretical topic this learning requires actually building system activities where we made the chatbot by using blocks and **_CSS_** language. This learning became more interesting because I could see the outcome of what I created. 

This was a completely new topic for me. I had not even heard about the term chabot before. So learning this at the start was way difficult, then I thought for this I took help form youtube and began exploring the tools that are there in typebot to understand the system properly. 

I can directly connect this with apps or websites which use chatbot for providing information or helping to solve problems. Now I actually know how this chabot is made.

The challenge I faced during this was the chatbot flow. Managing multiple blocks and ensuring that the conversation remained smooth and logical. It was difficult to keep changing things because keeping the track of all the blocks was confusing. 

The one important thing I learned is that we should always plan first about how we are going to build the structure of a chatbot before directly starting with the tools. Another thing that worked well was experimenting with design, tools and customizing by using theme and css language to make it look more professional and creative. 
**Overall this project helped me to understand how it is really important to make the boring conversation more creative and interactive for the user rather than using just a lot of information paragraphs.**
